

## 🌟 Advanced-Light-Intensity-Monitoring-System-ALIMS

The ALIMS is a hardware-based intelligent light monitoring system that captures, processes, and displays real-time and averaged light intensity using an LDR sensor. The system integrates analog filtering, digital processing, stability control, and visual output through dual seven-segment displays, implemented using logic gates and flip-flop-based design.

## 🔍 Overview

The Smart Light Intensity Monitoring System (SLIMS) is designed to provide accurate and stable measurement of ambient light levels. The system uses an LDR sensor to detect light intensity and converts it into an electrical signal. This signal is processed through filtering and digital logic modules to remove noise and improve reliability.

A key feature of this system is its **stability control mechanism**, which ensures that sudden fluctuations or noise do not affect the displayed output. The system calculates both real-time and averaged light intensity values, making it suitable for environments where consistent monitoring is essential.

---

## 🏗️ System Architecture

• **Light Sensor (LDR):** Converts ambient light into an analog voltage signal.

• **Signal Conditioning & Filtering:** Smooths the signal and removes noise using a second-order low-pass filter (Sallen-Key).

• **Flash ADC:** Converts analog signals into digital form for processing.

• **Processing Unit (Digital Logic):** Performs decision-making, averaging, and control operations.

• **Averaging Module:** Computes average light intensity over time.

• **Stability Logic Module:** Eliminates false triggers due to sudden light changes.

• **Display Unit:** Dual seven-segment displays show real-time and average values.

• **Power Supply:** Ensures stable and regulated system operation.

---

## ⚙️ How It Works (Brief)

The LDR detects ambient light and generates a proportional analog voltage.

This signal is passed through a filtering stage to reduce noise and fluctuations.

A flash ADC converts the analog signal into digital data for processing.

The digital logic unit calculates real-time and average light intensity values.

The stability module ensures output updates only when consistent changes occur.

Finally, the processed data is displayed using dual seven-segment displays for clear visualization.
